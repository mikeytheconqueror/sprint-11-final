Backend-сервис на Go для управления и отслеживания посылок.
Проект реализован в учебных целях для практики backend-разработки, работы с базой данных, тестирования и контейнеризации

Функциональность

Работа с сущностями посылок
Базовая бизнес-логика обработки данных
CRUD-операции
Unit-тесты для проверки корректности логики
Контейнеризация приложения с помощью Docker

Стек технологий

Go
SQLite
Docker
Testing (unit-tests)

Запуск проекта локально
go run main.go

Запуск тестов
go test ./...

Запуск через Docker
docker build -t parcel-tracker .
docker run parcel-tracker
